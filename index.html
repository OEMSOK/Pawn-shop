កាលបរិច្ឆេទ លេខសំបុត្រ អតិថិជន ប្រភេទ ប្រាក់បញ្ចាំចេញ ប្រាក់ដើមលោះចូល ការប្រាក់ប្រមូលបាន

រក្សាទុក និងស្ដារទិន្នន័យ

ទិន្នន័យទាំងអស់ត្រូវបានរក្សាទុកនៅលើ browser ក្នុងម៉ាស៊ីននេះផ្ទាល់។

ប្ដូរពណ៌ផ្ទៃប្រព័ន្ធ

ជ្រើសរើសប្រធានបទពណ៌ដែលអ្នកពេញចិត្តសម្រាប់ផ្ទៃប្រព័ន្ធទាំងមូល។

ការកំណត់ការភ្ជាប់ទៅ Cloud

ភ្ជាប់ប្រព័ន្ធទៅកាន់ Firebase Realtime Database ដើម្បីធ្វើសមកាលកម្ម (Sync) ទិន្នន័យរវាងឧបករណ៍ច្រើនក្នុងពេលតែមួយ។

សំគាល់ / Notice: ១. ដើម្បីដំណើរការបានដោយរលូន សូមកំណត់ Database Rules ទៅជា ".read": "true" និង ".write": "true" នៅក្នុង Firebase Console។ ២. ពេលបើក Cloud Sync ប្រព័ន្ធនឹងទាញយកទិន្នន័យពី Cloud (ប្រសិនបើមាន) ឬបញ្ជូនទិន្នន័យពីម៉ាស៊ីននេះទៅ Cloud (ប្រសិនបើ Cloud ទទេ)។
